# Loot Tracker

A game inspired by Oldschool Runescape and its predecessor Devious Mud


# Goals of this simple game

* You can pick one of three classes attacker (melee), caster (magic/ranged), healer (a tanky melee class but ability to heal and trades off damage for surviability -- in single play this might be the easy mode at the cost of taking longer to kill)

* Single player focus and a random world with simple graphics and using fonts to render text

* Future might include LAN and group play or player hosted servers...but don't quote me


# Game play loop

Kill, loot, repeat, with the loot basically improving your character, healing mostly coming from rare shops that sell it for gold, loot will be class based so some loot has to be sold you cannot switch classes
